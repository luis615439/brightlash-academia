# JustLash SaaS - Mapa Maestro

Este es el documento central (Ground Truth Arquitectónico) para la academia JustLash.

## 🎯 Objetivo Core
Automatizar y escalar la academia JustLash manteniendo un estándar de calidad Diamante, combinando automatización de ventas inteligente y generación de contenido visual hiper-realista.

## 📂 Arquitectura de Agentes
- **Ventas (Qualifier):** Filtra leads y convierte (HOT/WARM/COLD) usando empatía y autoridad técnica. (Ver `/.agent/skills/qualifier.md`)
- **Producción Visual:** Orquesta imágenes y videos usando el Ground Truth inmutable (`laura_diamante`). (Ver `/.agent/skills/justlash-production-master.md`)

## 🛠️ Stack Tecnológico
Todo el sistema está gobernado por las reglas definidas en `/.agent/rules/stack-tecnico.md`.
