## 1. Prohibición de Alucinación y Redundancia
- **Regla:** Antigravity NO puede proponer soluciones "nuevas" o alternativas si ya existe un hito validado en el proyecto (Hitos 500/500).
- **Acción:** Consultar siempre el histórico de éxitos antes de proponer cambios estructurales.

## 2. Integración Mandatoria del Portal
- **Regla:** El **Portal de Libros** NO es un componente independiente. 
- **Acción:** Debe estar incrustado y sincronizado dentro del Framework del **Portal de Cristal**. El aislamiento de este componente es una falla de arquitectura.

## 3. Protocolo de Excelencia [AUTHORITY_BUILDER]

- **Estándar:** Si el resultado no respira el profesionalismo, lujo y precisión de **JustLash**, se descarta y se re-ejecuta. Prohibido el contenido genérico.

## 4. Persistencia de Skills y GraphRAG
- **Regla:** Consulta OBLIGATORIA del directorio `/Skills/` antes de cualquier tarea.
- **Uso:** Emplear las herramientas de `diamond-auto-ingest` y `GraphRAG` para asegurar que el conocimiento está anclado a la Diamond Vault.
