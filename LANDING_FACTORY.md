# Sistema de Generación de Landings "One Shot"

## Instrucción de Activación
Cada vez que el usuario pida una "Landing", NO generes código de inmediato. Sigue este protocolo:

1. Presenta este cuestionario al usuario:
   - **Proyecto:** ¿Cómo se llama la marca?
   - **Producto/Servicio:** ¿Qué vendemos exactamente?
   - **Audiencia:** ¿A quién va dirigido?
   - **Propuesta de Valor:** ¿Qué nos hace diferentes/premium?
   - **Prueba Social:** ¿Algún dato de confianza?
   - **Call to Action:** ¿Qué botón queremos al final?
   - **Estilo Visual:** [Boutique Cinematográfico | Tech Moderno | Minimalista Clínico]

2. Una vez respondidas las preguntas, genera el archivo `index.html` siguiendo la estructura de alta fidelidad:
   - Header con Navbar estable (según PORTAL_CONTRACTS.md).
   - Hero con imagen de impacto y tipografía Serif/Sans premium.
   - Sección de Problema vs. Solución.
   - Grid de Servicios con micro-interacciones.
   - Footer con efecto "Glow" en el botón de contacto.
